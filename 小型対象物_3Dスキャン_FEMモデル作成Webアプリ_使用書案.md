# 小型対象物3Dスキャン・FEMモデル作成Webアプリ 使用書（案）

## 1. はじめに

本プログラムは、机の上に載せられる程度の小型対象物をスマートフォンなどで撮影し、撮影画像または動画から3D形状を作成し、FEM解析に利用できるサーフェスモデルおよびメッシュモデルを生成するWebアプリです。

主な対象は、次のような単一部品または小型試験体です。

- 金属ブラケット
- 穴付きプレート
- 木材・樹脂・コンクリート試験体
- 3Dプリンタ製部品
- 小型架台
- 接合部模型
- 機械部品
- 小型の破損・変形部品

本プログラムでは、最初の撮影後に未撮影部分や再構成精度の低い部分を確認し、追加で撮影すべき方向や位置を画面上に表示します。

> **重要**
>
> 本プログラムが画像から生成する形状は、撮影時にカメラから見えた表面形状です。  
> 材料内部、閉じた空洞、接触面、板厚、内部補強、配筋、ボルト締結状態などは、画像だけでは正確に判断できません。必要に応じて寸法や解析条件を利用者が入力してください。

- 追加撮影位置の案内

- カメラ位置の推定

- 三角形サーフェスモデルの作成
- 背景および机面の除去
- 表面欠損部の検出
- 不要部分の削除
- サーフェスの穴埋め
- 水密形状の作成
- 平面、円筒面、穴などの候補検出

### 2.3 FEMモデル作成

- 四面体ソリッドメッシュの作成
- 全体要素寸法の設定

- FEMメッシュおよび解析データの出力

## 3. 初期版での対象範囲

### 3.1 推奨対象

初期版では、次の条件を満たす対象物を推奨します。

- 1人で移動できる大きさ
- おおむね10 cm～1 m程度
- 机または撮影台に載せられる
- 表面が硬く、撮影中に形状が変化しない
- 全周から撮影できる
- 単一材料または材料区分が明確
- 光を強く反射しない
- 透明または半透明でない
- 極端に細い部品が少ない
- 内部形状を解析に使わなくてよい

### 3.2 初期版で対応しないもの

- 複数部品間の接触解析
- ボルト締結の自動認識
- 溶接部の自動モデル化
- 接着部の自動モデル化
- 材料非線形解析
- 破壊解析
- 塑性解析
- 内部空洞の自動推定
- 透明物体
- 鏡面仕上げの物体
- 布、ゴム、ケーブルなど撮影中に変形するもの
- 極端に細い棒、針、ワイヤ
- 板厚や内部補強の画像からの自動判定

## 4. 推奨利用環境

### 4.1 撮影端末

- スマートフォン
- タブレット
- デジタルカメラで撮影した画像のアップロード

深度センサーやLiDARを搭載した端末は撮影補助に利用できますが、必須ではありません。

### 4.2 Webブラウザ

- Google Chrome
- Microsoft Edge
- Safari
- その他WebGLまたはWebGPUに対応したブラウザ

### 4.3 通信環境

動画および大量の画像をアップロードするため、安定したWi-Fiまたは有線通信環境を推奨します。

### 4.4 撮影場所

- 明るさが安定している
- 対象物の全周に移動できる
- 背景が複雑すぎない
- 対象物と背景の色が異なる
- 強い影や反射が少ない
- カメラの自動露出が大きく変化しない

## 5. 撮影前の準備

### 5.1 対象物の準備

1. 対象物の汚れや不要な付着物を取り除きます。
2. 撮影中に対象物が動かないように置きます。
3. 光沢が強い場合は、照明位置を調整します。
4. 必要に応じて、取り外し可能なマーカーを周囲に配置します。
5. 対象物の実寸を確認します。

### 5.2 スケール基準の準備

FEM解析用の形状では、実寸の設定が重要です。次のいずれかを準備してください。

- 寸法が分かっているマーカー
- 方眼板
- 既知寸法のスケールブロック
- 定規
- 対象物上の既知寸法
- Web画面で指定できる2点間の実測寸法

### 5.3 撮影台

次の条件を推奨します。

- 表面が平坦
- 対象物と色が異なる
- 強く反射しない
- 模様が単調すぎない
- 必要に応じて位置確認用マーカーを設置

## 6. 基本的な利用手順

本プログラムの基本的な操作は、次の順番です。

```text
プロジェクト作成
  ↓
対象物情報の入力
  ↓
第1回撮影
  ↓
暫定3Dモデル作成
  ↓
撮影不足部分の確認
  ↓
追加撮影
  ↓
最終3Dモデル作成
  ↓
背景・机面の除去
  ↓
サーフェス修正
  ↓
FEMメッシュ作成
  ↓
材料・支持・荷重設定
  ↓
解析実行
  ↓
結果確認・出力
```

## 7. プロジェクトの作成

### 7.1 新規プロジェクト

1. トップ画面で **［新規プロジェクト］** を選択します。
2. 次の項目を入力します。

| 項目 | 内容 |
|---|---|
| プロジェクト名 | 対象物を識別できる名称 |
| 対象物名称 | 部品名、試験体名など |
| 単位 | mm、cm、m |
| おおよその大きさ | 幅、高さ、奥行き |
| 材料 | 鋼、アルミ、樹脂、木材、コンクリートなど |
| 解析目的 | 静解析、固有値解析など |
| 撮影方法 | 動画、静止画 |
| スケール設定 | マーカー、既知寸法、後入力 |

3. **［プロジェクトを作成］** を選択します。

### 7.2 推奨プロジェクト名

```text
対象物名_撮影日_管理番号
```

例：

```text
L型ブラケット_2026-07-12_TEST01
```

## 8. 第1回撮影

### 8.1 撮影開始

1. プロジェクト画面で **［撮影を開始］** を選択します。
2. 対象物全体が画面内に入る位置まで移動します。
3. 画面に表示される撮影ガイドに従います。
4. **［録画開始］** を選択します。
5. 対象物の周囲をゆっくり移動します。

### 8.2 推奨撮影方法

次の順番を推奨します。

#### 1周目：中間高さ

- 対象物の中心と同じ程度の高さ
- 対象物を中央に保持
- 一定の距離を保つ
- 周囲を1周する

#### 2周目：上方

- 対象物を斜め上から撮影
- 上面と側面が同時に見える角度
- 周囲を1周する

#### 3周目：下方

- カメラ位置を少し下げる
- 底面近くの側面や凹部を撮影
- 必要な範囲だけ撮影する

### 8.3 撮影中の注意

- 急にカメラを動かさないでください。
- 対象物との距離を大きく変えないでください。
- ズーム操作はできるだけ使用しないでください。
- 対象物が画面外に出ないようにしてください。
- 同じ場所で長時間停止する必要はありません。
- 反射が強い部分は、角度を変えて複数回撮影してください。
- 凹部や穴は、内部が見える方向から追加撮影してください。

---

## 9. 撮影中に表示される案内

撮影中、画面に次のような案内が表示されます。

| 表示例 | 意味 |
|---|---|
| ゆっくり移動してください | カメラ移動が速すぎます |
| 対象物に近すぎます | 一部が画面外になる可能性があります |
| 対象物から離れすぎています | 表面解像度が不足しています |
| 右側面が不足しています | 右側方向からの撮影が不足しています |
| 上面を撮影してください | 上面の観測数が不足しています |
| ブレを検出しました | 動画フレームが不鮮明です |
| 反射が強すぎます | 表面形状を認識しにくい状態です |
| この方向は撮影済みです | 追加撮影の効果が小さい方向です |
| 対象物を画面中央に入れてください | 対象物が追跡範囲から外れています |

## 10. 第1回撮影後の確認

撮影を終了すると、暫定的な3Dモデルと撮影品質マップを作成します。

### 10.1 撮影品質の色分け

| 色 | 状態 |
|---|---|
| 緑 | 十分に撮影されています |
| 黄 | 撮影枚数または角度が不足しています |
| 赤 | 未撮影または再構成できていません |
| 紫 | ブレ、反射、白飛びなどで再撮影が必要です |
| 灰 | 形状の有無を判定できていません |

### 10.2 表示の切り替え

次の表示を切り替えられます。

- 写真的3D表示
- 点群表示
- 暫定サーフェス
- 撮影品質ヒートマップ
- カメラ位置
- 撮影方向マップ
- 元画像
- 対象物マスク

### 10.3 不足箇所の確認

赤色または黄色の領域を選択すると、次の情報が表示されます。

- 不足している撮影方向
- 推奨するカメラ高さ
- 推奨する距離
- 推奨する撮影角度
- 追加撮影が必要な理由
- 推奨撮影位置までの移動方向

## 11. 追加撮影

### 11.1 追加撮影の開始

1. 撮影品質画面で不足箇所を選択します。
2. **［追加撮影を開始］** を選択します。
3. 画面上の矢印に従って移動します。
4. 推奨位置に到達したら、対象部分を数秒間撮影します。
5. 次の推奨位置が表示された場合は、続けて移動します。
6. 必要な箇所を撮影したら **［追加撮影を終了］** を選択します。

### 11.2 追加撮影の案内例

- 左へ30 cm移動してください
- カメラを約20 cm上げてください
- 対象物の裏側へ移動してください
- 穴の中心が見える位置に移動してください
- 面に対して正面に近い方向から撮影してください
- 現在位置から左右へ移動して視差を確保してください
- 反射が少なくなるよう撮影角度を変えてください

### 11.3 追加撮影の終了条件

次のいずれかを満たした場合、追加撮影を終了できます。

- 重要領域がすべて緑色になった
- 未撮影部分が解析に不要である
- 対象物の構造上、撮影できない部分である
- 利用者が不足部分を手動補完することを選択した

---

## 12. 底面の撮影

机上に置いた対象物では、底面を撮影できません。

### 12.1 底面撮影の手順

1. 通常向きの撮影を完了します。
2. **［別姿勢の撮影を追加］** を選択します。
3. 対象物を裏返します。
4. 底面および周辺を撮影します。
5. 通常向きの撮影データと位置合わせします。

### 12.2 位置合わせ方法

- 対象物表面の共通特徴
- 対象物に貼付したマーカー
- 利用者が指定した3点以上の対応点
- 既知寸法
- 点群の自動位置合わせ

### 12.3 底面を撮影しない場合

底面が単純な平面である場合は、次の方法で補完できます。

- 切断平面で閉じる
- 既知の高さを入力する
- 底面位置を数値指定する
- CADプリミティブとして平面を追加する

## 13. 最終3Dモデルの作成

追加撮影後、**［最終モデルを作成］** を選択します。

作成される主なデータは次のとおりです。

### 13.1 写真的3Dモデル

撮影時の色や質感を確認するためのモデルです。

用途：

- 撮影状態の確認
- 元画像との比較
- 欠損部分の確認
- FEM形状との比較
- 損傷や表面状態の記録

### 13.2 点群モデル

空間上の点の集合として表現した形状です。

用途：

- 寸法確認
- ノイズ除去
- サーフェス作成
- 別撮影データとの位置合わせ

### 13.3 サーフェスモデル

三角形面で構成された表面形状です。

用途：

- FEMメッシュ作成
- 面分解
- 穴埋め
- 平滑化
- 水密性確認

---

## 14. 背景と机面の除去

### 14.1 自動除去

**［対象物を抽出］** を選択すると、次の領域を自動検出します。

- 机面
- 背景
- 撮影マーカー
- 対象物から離れた点
- 浮遊ノイズ

### 14.2 手動修正

自動抽出結果が不十分な場合は、次の操作を行います。

- ブラシで削除
- 矩形で範囲削除
- 切断平面で削除
- 選択領域を対象物として保持
- 削除した領域を元に戻す

### 14.3 注意

対象物の底面と机面が一体化している場合、境界を自動判定できないことがあります。必要に応じて、底面位置を利用者が指定してください。

---

## 15. サーフェスの確認と修正

### 15.1 自動チェック

サーフェス作成後、次の項目を自動確認します。

- 開いた境界
- 欠損穴
- 重複面
- 反転面
- 自己交差
- 非多様体エッジ
- 浮遊面
- 極端に小さい面
- 極端に細長い三角形
- 内部に閉じ込められた不要面

### 15.2 修正機能

- 穴埋め
- ノイズ除去
- スムージング
- 稜線保持
- 面の削除
- 面の作成
- 面の分割
- 面の結合
- 平面への置換
- 円筒面への置換
- 対称化
- 寸法指定
- 切断
- 閉じたソリッドへの変換

### 15.3 平滑化の注意

過度な平滑化を行うと、穴、角部、溝、損傷形状などが失われます。平滑化前後の形状を比較しながら設定してください。

## 16. 面分解

本プログラムは、サーフェスから次の形状候補を抽出します。

- 平面
- 円筒面
- 球面
- 円錐面
- 自由曲面
- 穴の内周面
- フィレット
- 稜線
- 支持面候補
- 荷重面候補

### 16.1 面分解結果の修正

- 隣接する面を結合
- 1つの面を分割
- 面の種類を変更
- 平面位置を数値指定
- 円筒軸と直径を修正
- 小さな凹凸を無視
- FEM解析に不要な形状を削除

### 16.2 実測寸法による修正

撮影形状よりも実測値を優先する場合は、次の寸法を入力できます。

- 全幅
- 全高
- 奥行き
- 板厚
- 穴径
- 穴間隔
- フィレット半径
- 面間距離
- 円筒直径

## 17. FEM用形状の作成

### 17.1 初期版のモデル

初期版では、単一物体の四面体ソリッドモデルを作成します。

```text
修正済みサーフェス
  ↓
水密性確認
  ↓
閉じた体積領域
  ↓
四面体FEMメッシュ
```

### 17.2 解析形状の理想化

撮影形状には、表面粗さ、傷、小さな凹凸、撮影ノイズが含まれます。必要に応じて、解析前に次の処理を行います。

- 小さな傷を除去
- 表面を平面化
- 円筒面を真円化
- 左右対称化
- 小さな穴を除去
- フィレットを単純化
- 解析に不要な刻印や文字を除去

### 17.3 元形状との比較

理想化前後を重ね合わせて確認できます。

- 撮影点群
- 生サーフェス
- 修復済みサーフェス
- FEM用形状

解析形状が実物と大きく異なっていないことを確認してください。

## 18. FEMメッシュの作成

### 18.1 基本設定

**［FEMメッシュ作成］** 画面で次の項目を設定します。

| 項目 | 内容 |
|---|---|
| 要素形式 | 四面体 |
| 要素次数 | 一次または二次 |
| 全体要素寸法 | モデル全体の標準要素寸法 |
| 最大要素数 | 生成を許可する要素数 |
| 曲率細分化 | 曲面部分を細かく分割 |
| 局所細分化 | 選択面や穴周辺を細かく分割 |
| 品質下限 | 使用可能とする要素品質 |

### 18.2 推奨設定

最初は、対象物の代表寸法に対して次の程度から開始します。

- 全体要素寸法：代表寸法の1/20～1/50
- 穴周辺：全体要素寸法の1/2程度
- フィレット周辺：必要に応じて局所細分化
- 二次要素：曲面や応力評価を重視する場合

### 18.3 メッシュ品質確認

次の項目を確認します。

- 負体積要素がない
- 極端に細長い要素が少ない
- 小さすぎる要素がない
- メッシュが分断されていない
- 穴や凹部が正しく表現されている
- 支持面と荷重面が保持されている
- 要素数が計算可能な範囲にある

---

## 19. 材料の設定

### 19.1 基本材料定数

線形静解析および固有値解析では、主に次の項目を設定します。

- ヤング係数
- ポアソン比
- 密度
- 材料名称

### 19.2 材料データベース

材料データベースから一般的な値を選択できますが、実際の解析では対象物の仕様、試験値、設計条件に合わせて確認してください。

### 19.3 注意

画像から材料定数を自動的に決定することはできません。外観上の材料候補が表示された場合でも、最終的な材料設定は利用者が確認してください。

## 20. 支持条件の設定

### 20.1 支持面の選択

1. **［支持条件］** を選択します。
2. 3D画面上で支持する面を選択します。
3. 拘束する方向を指定します。

設定例：

- 完全固定
- X方向拘束
- Y方向拘束
- Z方向拘束
- 法線方向拘束
- 指定点の拘束

### 20.2 注意

撮影時に机へ接していた面が、実際の解析でも固定面になるとは限りません。実際の設置条件、接触条件、治具条件に合わせて設定してください。

## 22. 解析の実行

### 22.1 線形静解析

変位、応力、ひずみ、反力などを確認します。

1. 材料を設定します。
2. 支持条件を設定します。
3. 荷重条件を設定します。
4. **［線形静解析を実行］** を選択します。
5. 計算終了後、結果画面を確認します。

### 22.2 固有値解析

固有振動数と固有モードを確認します。

1. 材料のヤング係数、ポアソン比、密度を設定します。
2. 支持条件を設定します。
3. 求めるモード数を入力します。
4. **［固有値解析を実行］** を選択します。

- 主応力
- ひずみ
- 反力
- 固有振動数

### 23.2 表示倍率

### 23.3 最大値の扱い

最大応力値だけでなく、次の内容を併せて確認してください。

- 応力分布
- メッシュ密度
- 支持条件付近
- 荷重作用位置
- 小さな凹凸の有無
- 理想化モデルとの比較

## 25. プロジェクトの保存と履歴

本プログラムでは、次の段階を別データとして保存します。

```text
撮影データ
  ↓
暫定再構成
  ↓
最終再構成
  ↓
生サーフェス
  ↓
修復済みサーフェス
  ↓
FEM用形状
  ↓
FEMメッシュ
  ↓
解析条件
  ↓
解析結果
```

元データを上書きせず、各段階を履歴として保持します。

### 25.1 保存される主な履歴

- 撮影日時
- 追加撮影日時
- 形状修正内容
- 寸法修正内容
- メッシュ設定
- 材料変更
- 支持条件変更
- 荷重条件変更
- 解析実行履歴

## 26. 撮影品質を高めるためのポイント

### 26.1 表面に模様が少ない場合

単色で模様がない物体は、画像間の対応点を取得しにくい場合があります。

対策：

- 対象物の周囲にマーカーを置く
- 取り外し可能な小さな模様を付ける
- 背景に特徴のある模様を配置する
- 照明を斜めから当てる
- 深度センサーを併用する

### 26.2 金属表面

対策：

- 強い直接照明を避ける
- 複数方向から柔らかく照明する
- 反射が変化する方向から複数回撮影する
- 必要に応じて表面処理を検討する
- 実物を変更できない場合は、反射領域を手動補正する

### 26.3 凹部や穴

対策：

- 正面だけでなく斜め方向から撮影する
- 穴の中心線に近い方向から撮影する
- カメラを近づける
- 照明を追加する
- 必要に応じて静止画を追加する

### 26.4 細い部分

対策：

- 背景との色差を大きくする
- 高解像度で撮影する
- カメラを近づける
- 複数方向から撮影する
- メッシュ作成時に局所細分化する

## 27. 警告メッセージ

### 27.1 撮影関係

| 警告 | 対応 |
|---|---|
| 撮影方向が不足しています | 指定方向から追加撮影します |
| カメラ姿勢を推定できません | 模様のある背景やマーカーを追加します |
| 動画にブレが多く含まれます | ゆっくり撮り直します |
| 対象物が画面外です | 対象物を中央に入れます |
| 露出差が大きすぎます | 照明と自動露出を調整します |
| 反射領域を再構成できません | 角度または照明を変更します |

### 27.2 形状関係

| 警告 | 対応 |
|---|---|
| サーフェスが閉じていません | 欠損部を撮影または穴埋めします |
| 自己交差があります | 問題領域を修正します |
| 机面との境界が不明です | 切断平面を指定します |
| 形状寸法が設定されていません | 基準寸法を入力します |
| 底面が不足しています | 裏返し撮影または平面補完します |

### 27.3 FEM関係

| 警告 | 対応 |
|---|---|
| 四面体メッシュを作成できません | サーフェスの水密性を確認します |
| 要素品質が低い領域があります | 局所形状または要素寸法を修正します |
| 支持条件がありません | 支持面を設定します |
| 材料が設定されていません | 材料定数を入力します |
| 剛体移動が発生しています | 拘束条件を確認します |
| 要素数が上限を超えています | 要素寸法を大きくします |

## 28. トラブルシューティング

### 28.1 3Dモデルが一部しか作成されない

考えられる原因：

- 撮影方向が偏っている
- 動画の移動が速い
- 表面に模様がない
- 反射が強い
- 対象物が小さく映っている
- 画像が暗い

対応：

- 撮影品質マップを確認する
- 赤色部分を追加撮影する
- カメラを近づける
- マーカーを追加する
- 照明を調整する

### 28.2 形状が膨らむ、へこむ

考えられる原因：

- 視差不足
- 撮影距離が一定でない
- 反射
- 背景と対象物の分離不良
- 平滑化の過剰適用

対応：

- 異なる方向から追加撮影する
- 生点群とサーフェスを比較する
- 平面または円筒へ置換する
- 実測寸法を入力する

### 28.3 穴がふさがる

考えられる原因：

- 穴内部の撮影不足
- サーフェス生成時の自動穴埋め
- 平滑化の影響

対応：

- 穴の正面および斜め方向から追加撮影する
- 自動穴埋めを解除する
- 円筒面として穴を再作成する
- 実測した穴径を入力する

### 28.4 FEMメッシュが細かすぎる

対応：

- 全体要素寸法を大きくする
- 表面ノイズを除去する
- 不要な小形状を削除する
- 局所細分化の設定を見直す

### 28.5 最大応力が極端に大きい

確認事項：

- 点荷重になっていないか
- 支持条件が局所的すぎないか
- 小さな凹凸が残っていないか
- メッシュ品質が低くないか
- 応力特異点ではないか

## 29. 解析結果を利用する際の注意

本プログラムは、撮影形状からFEMモデル作成を支援するものです。解析結果の妥当性は、次の条件に依存します。

- 寸法精度
- 撮影できない部分の補完方法
- 材料定数
- 支持条件
- 荷重条件
- 接触条件
- 形状の理想化
- メッシュ品質
- 解析方法

特に、画像から直接確認できない内部構造や接合状態については、図面、仕様書、実測、試験結果などを用いて補完してください。

## 31. 推奨する検証方法

同じ対象物について、次の3モデルを比較します。

```text
A：元のCAD形状から作成したFEMモデル
B：撮影形状をそのまま修復して作成したFEMモデル
C：撮影形状を平面・円筒などへ理想化したFEMモデル
```

比較項目：

- 寸法
- 体積
- 要素数
- 最大変位
- 応力分布
- 固有振動数
- モデル作成時間
- 手動修正時間

## 35. 免責事項（案）

本プログラムで生成される3D形状、FEMメッシュおよび解析結果は、撮影条件、画像品質、形状補完、材料設定、境界条件、荷重条件などにより変化します。

本プログラムの解析結果のみを根拠として、製品の安全性、構造性能、耐久性、破壊荷重などを最終判断しないでください。重要な設計判断に使用する場合は、元図面、実測、材料試験、既存解析、実験結果および専門技術者による確認を併用してください。
